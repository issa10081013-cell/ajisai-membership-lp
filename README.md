## 🛠️ トラブルシューティング

### **Q: ペライチでセクション追加時にレイアウトが崩れる**
→ ペライチの「プレビュー」で確認しながら調整してください。セクション幅は「フル幅」がおすすめです。

### **Q: Google Forms のリンクが長い（短くしたい）**
→ Google Forms の「送信」ボタン > リンクタブ > 短いリンク（https://forms.gle/...）をコピーしてください。

### **Q: Instagram の画像サイズがわからない**
→ 推奨：**1080 × 1350px**（インスタグラムの標準サイズ）。Canva で簡単に作成できます（無料テンプレート豊富）。

### **Q: LINE 公式で複数メッセージを同時送信できない**
→ 1回の配信で複数メッセージを送るには、メッセージをまとめて「1つのセット」として送信してください。ペライチでは可能ですが、LINE 公式では分割配信される場合があります。

### **Q: Google Forms の回答が Google Sheet に反映されない**
→ フォーム設定で「リンク済みスプレッドシートを自動作成」を確認してください。

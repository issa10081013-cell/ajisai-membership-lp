# Google Forms 会員登録フォーム セットアップガイド

---

## 【事前準備】

- Googleアカウント（Gmail）
- 10分程度の時間

---

## 【STEP 1】Google Forms を作成する

1. https://forms.google.com にアクセス
2. 「＋（新規作成）」ボタンをクリック
3. 「空白」を選択

---

## 【STEP 2】フォームのタイトル＆説明を設定

**タイトル:** あじさい 会員登録フォーム

**説明文:**
```
ようこそ！あじさいの会員登録フォームです。

以下の情報を入力すると、
メールにて会員情報と初期ログイン情報をお送りします。

初回7日間は無料トライアル可能です。
ご不明な点はお気軽にお問い合わせください。
```

---

## 【STEP 3】設問を追加する

### 質問 1: 名前
- **質問タイプ:** 短編テキスト
- **質問:** お名前（フルネーム）
- **必須:** ✓ チェック

### 質問 2: メールアドレス
- **質問タイプ:** メール
- **質問:** メールアドレス
- **説明:** ログイン情報をこちらにお送りします
- **必須:** ✓ チェック

### 質問 3: 電話番号
- **質問タイプ:** 短編テキスト
- **質問:** 電話番号
- **説明:** お問い合わせ時に使用します
- **必須:** ✓ チェック

### 質問 4: 会員プラン選択
- **質問タイプ:** 複数選択
- **質問:** どのプランをお選びになりますか？
- **選択肢:**
  - スターター（月¥1,000）- ロールモデル動画視聴 + 月1回ライブ配信
  - スタンダード（月¥3,000）- 週1～2回ライブ配信 + 親向けセミナー月2回
  - VIP（月¥5,000）- すべて + 個別相談月1回 + 特別割引
- **必須:** ✓ チェック

### 質問 5: 子どもの学年（任意）
- **質問タイプ:** ドロップダウン
- **質問:** お子様の学年をお知らせください（任意）
- **選択肢:**
  - 小学1年生
  - 小学2年生
  - 小学3年生
  - 小学4年生
  - 小学5年生
  - 小学6年生
  - 中学1年生
  - 中学2年生
  - 中学3年生
  - 高校生
  - その他
- **必須:** チェック不要

### 質問 6: ご質問・ご相談（任意）
- **質問タイプ:** 長編テキスト
- **質問:** ご質問やご相談があればお知らせください
- **説明:** こちらにいただいたお問い合わせにはメールでご返信させていただきます
- **必須:** チェック不要

---

## 【STEP 4】フォーム設定を調整する

1. 画面右上の **「⚙️（設定）」** をクリック
2. **「フォーム」** タブで以下を設定:

| 設定項目 | 設定値 |
|---------|--------|
| メールアドレスを収集する | ON |
| ログインが必要 | OFF（誰でも応募可能に） |
| 回答を編集できるようにする | ON（修正可能に） |
| 回答が提出されたときに通知メールを受け取る | ON |
| リンクされたスプレッドシートを自動作成 | ON（自動集計用） |
| プログレスバーを表示 | ON（ユーザー体験向上） |
| 過去の回答を表示 | OFF |
| 回答のシャッフル | ON（質問順序をランダム化） |

---

## 【STEP 5】フォームのデザイン・カラーを変更する

1. 画面上部の **「パレット（デザイン）」** をクリック
2. テーマを選択:
   - 推奨：**「カスタム」** を選び、以下に設定
   - **ヘッダー色:** 紫（#5B3FA0）
   - **フォント:** Arial / Google Sans
   - **背景:** 白（#FFFFFF）

---

## 【STEP 6】送信＆リンク取得

1. 画面右上の **「送信」** ボタンをクリック
2. **「リンク」** タブを選択
3. 短いリンクを作成:
   - 【例】 https://forms.gle/xxxx...（自動生成）

4. このリンクをコピーして、ペライチのCTAボタンに貼り付け

---

## 【STEP 7】自動返信メールを設定（あると便利）

1. 回答 > 回答の概要 > **「自動メール通知を有効にする」**
2. **「ユーザーメール機能を有効にする」** にチェック
3. 返信メールの件名:
   ```
   あじさい 会員登録ありがとうございました！
   ```
4. 返信メール本文:
   ```
   【あじさい】会員登録ありがとうございます！

   お選びいただいたプラン: [プラン名]
   会員開始日: 本日から7日間無料トライアル
   
   ■ ご注意ください ■
   このメールはお客様がご入力いただいたメールアドレスに自動送信されています。
   
   ■ 次のステップ ■
   1. メンバーサイトにアクセス: https://ajisai-member.com
   2. ログインID: [メールアドレス]
   3. パスワード: お別途ご連絡します
   
   ご不明な点やログインできない場合は、
   こちらのチャット機能からお問い合わせください。
   
   【LINE公式でも情報配信中】
   https://line.me/R/ti/p/@ajisai
   
   よろしくお願いいたします。
   あじさい 一同
   ```

---

## 【STEP 8】スプレッドシートで回答を管理

1. Google Forms の **「回答」** タブ
2. **「スプレッドシート」** アイコンをクリック
3. 自動的に Google Sheet が作成され、回答が記録される
4. シート上で以下を管理:
   - タイムスタンプ
   - 回答者メール
   - プラン別集計
   - フォローアップ必要な質問

---

## 【トラブルシューティング】

**Q: フォームのリンクが長い**
→ Google Forms の「送信」で短いリンク（https://forms.gle/...）を取得できます

**Q: 回答がメールで届かない**
→ 「フォーム」設定の「回答が提出されたときに通知メールを受け取る」をONにしてください

**Q: スプレッドシートに自動反映されない**
→ 「フォーム」設定で「リンクされたスプレッドシートを自動作成」をONにしてください
